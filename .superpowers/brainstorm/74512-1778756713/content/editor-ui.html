交互流程

1. 首页：上传文件 → 选择主题 → 点击「生成」
2. 编辑器：
   - 左栏：幻灯片缩略图列表，支持拖拽排序、添加/删除
   - 中栏：iframe 实时预览，点击元素进入编辑状态
   - 右栏：样式面板（主题/布局/动效/内容编辑）
3. 点击幻灯片 → 右侧面板显示该页的属性
4. 修改任何属性 → iframe 自动刷新预览
5. 导出：生成完整 HTML 文件下载
