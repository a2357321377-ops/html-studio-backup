编辑器交互总结

  • 就地编辑：点击 iframe 中的文字 → contenteditable → 浮动工具条（加粗/斜体/字号/颜色/对齐）
  • 拖拽排序：左栏幻灯片支持拖拽重排，蓝色指示线显示插入位置
  • 布局切换：右侧面板选择布局 → 内容自动映射到新布局的插槽
  • 主题/动效：切换即时生效，iframe 自动刷新
  • 导出：生成完整独立 HTML（含 runtime.js + 主题 CSS + 动效），可选导出 JSON 源数据
  • 全屏预览：新标签页打开完整幻灯片，支持键盘导航
